• Native JS Slider

    Данный слайдер выполнен на чистом, нативном JavaScript, без использования библиотеки jQuery.

  • Построение DOM через JavaScript.

    Динамическое конструирование DOM дерева, обеспечивающее более безопасный и гибкий функционал для пользователя.

  • Реализован следующий функционал:

    • Автоматическая прокрутка слайдера
    • Управление кнопками на экране
    • Клавиатурное управление
    • Управление с помощью индикаторов
    • Управление свайпами, для мобильных устройств

  • Удобный для понимания код

    Для управления всем слайдером реализован главный конфиг
    в отдельном файле main.js
    Так же выполнена де структуризация и де композиция кода.

  • Продуманная файловая структура

    Воплощена как базовая версия, так и продвинутая (свайпы).
    За каждую версию отвечает отдельный JavaScript файл.
    Реализован модульный подход по работе со скриптами

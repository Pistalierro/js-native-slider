• Native JS Slider

    Данный слайдер выполнен на чистом, нативном JavaScript, без использования библиотеки jQuery.

  • Построение DOM дерева через JavaScript

    Динамическое конструирование DOM дерева, обеспечивающее более безопасный
    и гибкий функционал для пользователя.

  • В слайдере реализован следующий функционал

    • Автоматическая воспроизведение слайдера
    • Управление кнопками на экране
    • Клавиатурное управление
    • Управление с помощью индикаторов
    • Управление касанием для мобильных устройств
    • При клике на любое место экрана, слайдер либо останавливается, либо воспроизводится

  • Удобный для понимания код

    Для управления всем слайдером реализован главный конфиг в отдельном файле main.js
    Так же выполнена де структуризация и де композиция кода.
